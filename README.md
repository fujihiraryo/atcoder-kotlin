# AtCoder with Kotlin

## setup

A directory `ABC/ABC123` and files `A.kt`, ..., `F.kt` are created.

```bash
./setup.sh ABC/ABC123
cd ABC/ABC123
```
## make sample

```bash
./run.sh sampleA.kt > sampleA.txt
```

## run codes

```bash
./run.sh A.kt < sampleA.txt
```

## formatting

```bash
ktlint -F A.kt
```
